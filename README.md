# Mi Chatbot con IA 🤖

Un chatbot funcional construido desde cero con HTML, CSS y JavaScript, integrado con inteligencia artificial a través de la API de Groq.

## ¿Qué hace?

- Responde preguntas en tiempo real usando IA
- Interfaz de chat con burbujas animadas
- Indicador de "Escribiendo..." mientras procesa la respuesta
- Soporte para enviar mensajes con Enter o con el botón


## Tecnologías usadas

- HTML5
- CSS3
- JavaScript
- Groq API (plataforma de inferencia)
- LLaMA 3.1 8B de Meta (modelo de lenguaje)
